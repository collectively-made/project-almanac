## What We Built

Project Almanac went from an empty directory to a fully functional self-hosted survival knowledge platform in a single session.

### Core Platform
- **FastAPI backend** with SSE streaming, async LLM inference (ThreadPoolExecutor + Semaphore), SQLite with WAL mode
- **React frontend** (Vite + TypeScript) with Field Desk UI design — DM Mono/Sans typography, olive-black palette, amber/sage accents
- **RAG pipeline** — sqlite-vec (dense) + FTS5 (sparse) + Reciprocal Rank Fusion hybrid retrieval
- **470 knowledge chunks** from public domain sources: USDA canning guides (7 booklets), FEMA preparedness, wilderness survival (FM 3-05.70), first aid, homesteading, emergency preparedness
- **PDF-to-markdown pipeline** — pymupdf4llm for born-digital PDFs, build_packs.py for markdown→JSONL chunking

### Features
- **Conversation threads** with SQLite persistence, sidebar, auto-titling, delete
- **Conversation history** — multi-turn context (last 10 messages sent with each request)
- **User context profiles** — form editor (always available) + LLM conversational builder (7B+ models)
- **Model-aware routing** — detects model size, routes to form vs conversation for profile building
- **Sources accordion** — expandable excerpts from retrieved chunks, colored icons by source type
- **Confidence scoring** — retrieval quality → visual meter with explainer text
- **Auto-setup** — zero-config first boot, auto-downloads recommended model
- **Streaming responses** — SSE with token/done/error events, markdown rendering

### Infrastructure
- **Docker** — multi-stage build, non-root user, read-only filesystem, capability drop, HEALTHCHECK
- **CI/CD** — GitHub Actions builds multi-arch images (amd64 + arm64) on every push to main
- **Security hardening** — SSRF protection, path traversal fixes, CORS lockdown, history injection prevention, FTS5 sanitization, SQLite busy_timeout, generation timeout with cancellation

### Design
- "Field Desk" aesthetic — field manual meets modern chat app
- TextQL-inspired sources dropdown with expandable excerpts
- Terminal-style input with `>` prompt
- Suggested questions on empty state
- Privacy-first messaging throughout

## Architecture

```
Single Docker Container
├── FastAPI (Python 3.12)
│   ├── /api/chat — RAG + LLM streaming
│   ├── /api/threads — conversation CRUD
│   ├── /api/context/profile — user context
│   ├── /api/models — model management
│   ├── /api/setup — first-run + model download
│   └── /api/health — status
├── React SPA (served as static files)
├── sqlite-vec (vector search)
├── SQLite FTS5 (keyword search)
├── llama-cpp-python (local LLM)
└── fastembed (local embeddings)
```

## Security Review

Ran a 4-agent security review (Security Sentinel, Architecture Strategist, Performance Oracle, Python Reviewer). Found and fixed:
